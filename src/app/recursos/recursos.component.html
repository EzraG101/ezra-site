Recursos

Esta página contiene enlaces a varios recursos útiles para estudiar y practicar para olimpiadas de matemáticas. Si conocen algún recurso que no incluyo en esta página, ¡pueden contactarme para que lo agregue!

~Ezra Guerrero Alvarez

Recursos en Español
  • Problema 0 (Servidor de Discord)

    • Este servidor de Discord (creado por el gran Tomás Cantú) es una comunidad para aquellos que quieren aprender más matemática. ¡Es un lugar para trabajar con otras personas y aprender mucho!

  • Proyecto MURO (proyectomuro.com)

    • Este sitio contiene los enunciados en español de problemas de diversas olimpiadas.

  • Mi Sitio Web (ezrag.me)

    • Entre otras cosas, mi sitio contiene folletos y clases enfocados en matemáticas de olimpiada.

Recursos en Inglés
  • The Art of Problem Solving (artofproblemsolving.com)

    • Este sitio está repleto de problemas de matemática. También ofrecen clases y libros.

  • Sitio Web de Evan Chen (evanchen.cc)

    • En el sitio web de Evan Chen pueden encontrar gran cantidad de folletos y videos sobre diversos temas de matemáticas. Evan Chen también ofrece un programa de entrenamiento llamado OTIS que puede ser muy útil una vez se tenga un poco de experiencia.
